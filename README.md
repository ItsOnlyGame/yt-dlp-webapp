# yt-dlp app

This is a web app created in React and Go to serve as a web UI for yt-dlp.

## Requirements

- yt-dlp
- ffmpeg
- go
- bun

## Getting started

### Frontend

```bash
bun install
bun run dev
```

### Backend

```bash
go install
go run main.go
```
